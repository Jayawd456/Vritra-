# Interactive Calendar Component

A fully functional React wall-calendar inspired component built for a frontend engineering challenge.

## Features

- Wall calendar style layout with a prominent hero image
- Month navigation (`Prev`, `Today`, `Next`)
- Date range selection with clear start/end/in-between visual states
- Integrated notes section for:
  - selected day
  - selected range
  - current month (when no day/range is selected)
- Notes persisted in browser `localStorage`
- Responsive layout for desktop and mobile screens

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal (usually `http://localhost:5173`).

## Build for Production

```bash
npm run build
npm run preview
```
