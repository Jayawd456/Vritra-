🏥 VRITRA – AI-Powered Telemedicine Platform

VRITRA is a smart healthcare platform designed to provide real-time medical assistance, AI-based diagnosis, and emergency support. It connects patients and doctors seamlessly through modern web technologies.

🚀 Features
📹 Real-Time Video Consultation
Secure doctor-patient communication using WebRTC & Socket.io
🤖 AI-Based Diagnosis Tools
Posture Detection
Medical Report Analysis
Symptom Checker
🚨 Emergency Mode
One-click SOS alert
Live location tracking
Instant assistance system
💊 Medicine Comparator
Compare medicines based on price, availability, and alternatives
🏥 Hospital Locator
Find nearby hospitals with mapping integration
🔐 Secure & Scalable Backend
Built with Node.js, Express, and Firebase
🛠️ Tech Stack
Frontend: React.js, Tailwind CSS
Backend: Node.js, Express.js
Real-Time: WebRTC, Socket.io
Database: Firebase / Firestore
AI Integration: Python-based APIs
💻 Run Locally
npm install
npm run dev

Open the local URL shown in the terminal (usually http://localhost:5173).

📦 Build for Production
npm run build
npm run preview
🎯 Project Vision

VRITRA aims to make healthcare accessible, fast, and intelligent by combining AI with telemedicine. It is especially useful for remote areas and emergency situations where immediate medical attention is critical.
